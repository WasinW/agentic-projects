## 1. บทนำเกี่ยวกับคลื่นย้อนกลับ

### 1.1 คำจำกัดความ

คลื่นย้อนกลับ (Corrective Waves) เคลื่อนที่ **สวนทาง** กับทิศทางแนวโน้มในระดับชั้นที่ใหญ่กว่าถัดไป มันมีความซับซ้อน หลากหลาย และยากต่อการระบุแบบเรียลไทม์มากกว่าคลื่นแรงกระตุ้น (Impulse/Motive Waves) ในขณะที่คลื่นแรงกระตุ้นมีโครงสร้าง 5 คลื่นเสมอ การย้อนกลับมาในรูปแบบที่หลากหลายมาก

### 1.2 หลักการพื้นฐาน

> **การย้อนกลับไม่เคยมีห้าคลื่น** มีเพียงคลื่นขับเคลื่อน (Motive Waves) เท่านั้นที่มีห้าคลื่น การเคลื่อนที่ "ห้าคลื่น" ที่ปรากฏในทิศทางสวนแนวโน้ม อาจเป็นคลื่นขับเคลื่อนของแนวโน้มใหม่ หรือเป็นส่วนหนึ่งของการย้อนกลับที่ซับซ้อนกว่า

หลักการนี้สำคัญยิ่งสำหรับระบบ AI: หากการเคลื่อนที่สวนแนวโน้มแบ่งย่อยเป็นห้าคลื่น มันส่งสัญญาณว่าการย้อนกลับอาจสิ้นสุดลงแล้วและแรงกระตุ้นใหม่ในทิศทางสวนแนวโน้มได้เริ่มต้น หรือเป็นคลื่น A ของ Zigzag (ซึ่งมีโครงสร้าง 5-3-5)

### 1.3 ตำแหน่งที่การย้อนกลับปรากฏ

| ตำแหน่ง | ภายใน | วัตถุประสงค์ |
|---|---|---|
| คลื่น 2 | แรงกระตุ้น (Impulse) | ย้อนกลับคลื่น 1 |
| คลื่น 4 | แรงกระตุ้น (Impulse) | ย้อนกลับคลื่น 3 |
| คลื่น B | Zigzag, Flat | การเคลื่อนที่สวนทางภายในการย้อนกลับ |
| คลื่น X | การย้อนกลับซับซ้อน | คลื่นเชื่อมต่อระหว่างการย้อนกลับแบบง่าย |
| ทั้งชุด A-B-C | หลังแรงกระตุ้นเสร็จ | ย้อนกลับการเคลื่อนที่ 5 คลื่นทั้งหมด |

### 1.4 การจำแนกรูปแบบการย้อนกลับ

```
Corrective Patterns
├── Simple (Single) Corrections
│   ├── Zigzag (5-3-5)
│   │   ├── Standard Zigzag
│   │   └── Running Zigzag (rare)
│   ├── Flat (3-3-5)
│   │   ├── Regular Flat
│   │   ├── Expanded (Irregular) Flat
│   │   └── Running Flat
│   └── Triangle (3-3-3-3-3)
│       ├── Contracting Triangle
│       ├── Barrier Triangle
│       ├── Expanding Triangle (rare)
│       └── Running Triangle
│
└── Complex (Combination) Corrections
    ├── Double Zigzag (5-3-5)-(x)-(5-3-5)
    ├── Triple Zigzag (5-3-5)-(x)-(5-3-5)-(x)-(5-3-5)
    ├── Double Three WXY
    │   ├── Zigzag + Flat
    │   ├── Zigzag + Triangle
    │   ├── Flat + Zigzag
    │   ├── Flat + Triangle
    │   └── Flat + Flat
    └── Triple Three WXYXZ
        └── (Any combination of zigzag, flat, triangle)
```

## 2. การย้อนกลับแบบง่าย: Zigzag (5-3-5)

### 2.1 โครงสร้าง

Zigzag ประกอบด้วยสามคลื่นกำหนดป้าย **A-B-C** โดยที่:

| คลื่น | โครงสร้างภายใน | ทิศทาง |
|---|---|---|
| คลื่น A | แรงกระตุ้น 5 คลื่น | สวนแนวโน้มหลัก |
| คลื่น B | ย้อนกลับ 3 คลื่น | ย้อนกลับบางส่วนของ A |
| คลื่น C | แรงกระตุ้น 5 คลื่น | ทิศทางเดียวกับ A |

**จำนวนคลื่นย่อยรวม:** $5 + 3 + 5 = 13$ (เลข Fibonacci)

### 2.2 กฎและแนวทาง

**กฎ (ไม่อาจละเมิดได้):**
1. คลื่น A ต้องแบ่งย่อยเป็น 5 คลื่น (Impulse หรือ Leading Diagonal)
2. คลื่น B ต้องแบ่งย่อยเป็น 3 คลื่น (รูปแบบย้อนกลับใดก็ได้)
3. คลื่น C ต้องแบ่งย่อยเป็น 5 คลื่น (Impulse หรือ Ending Diagonal)
4. คลื่น B ไม่สามารถย้อนกลับเกิน 100% ของคลื่น A
5. คลื่น C ต้องเคลื่อนที่เกินจุดสิ้นสุดของคลื่น A

**แนวทาง (แนวโน้มที่พบ):**
| แนวทาง | ค่าทั่วไป | หมายเหตุ |
|---|---|---|
| การย้อนกลับคลื่น B ของ A | 38.2%--78.6% | พบบ่อยที่สุดคือ 50%--61.8% |
| คลื่น C เทียบกับ A | 61.8%--161.8% ของ A | พบบ่อยที่สุดคือ 100% (เท่ากัน) |
| คลื่น C เทียบกับ A (Forex) | 100%--127.2% ของ A | ความสัมพันธ์ที่สะอาด |
| คลื่น C เทียบกับ A (Crypto) | 100%--161.8% ของ A | มักเกินเป้า |

### 2.3 ความสัมพันธ์ Fibonacci

$$P_B = P_A + r_B \times (P_{\text{start}} - P_A) \quad \text{where } r_B \in \{0.382, 0.500, 0.618, 0.786\}$$

$$P_C = P_B - e_C \times |P_A - P_{\text{start}}| \quad \text{where } e_C \in \{0.618, 1.000, 1.272, 1.618\}$$

### 2.4 การเทรด Zigzag

**ชุดที่ 1: เทรดคลื่น C หลังคลื่น B เสร็จสมบูรณ์**
```
ENTRY:     Sell (Zigzag ขาลง) ที่จุดเสร็จสมบูรณ์คลื่น B (50%-61.8% ของ A)
STOP:      เหนือจุดเริ่มต้น Zigzag (เหนือคลื่น B หาก Stop แน่น)
TARGET:    จุดสิ้นสุดคลื่น C ที่ 100%-161.8% ของคลื่น A
RR RATIO:  2:1 ถึง 4:1
```

**ชุดที่ 2: เทรดจุดสิ้นสุดของ Zigzag (กลับตัว)**
```
ENTRY:     Buy ที่จุดเสร็จสมบูรณ์คลื่น C (จุดสิ้นสุด Zigzag)
STOP:      ใต้จุดสิ้นสุดคลื่น C - ATR buffer
TARGET:    คลื่นแรงกระตุ้นใหม่ในทิศทางแนวโน้มเดิม
RR RATIO:  3:1 ถึง 5:1+
```

## 3. การย้อนกลับแบบง่าย: Flat (3-3-5)

### 3.1 โครงสร้าง

Flat ประกอบด้วยสามคลื่นกำหนดป้าย **A-B-C** โดยที่:

| คลื่น | โครงสร้างภายใน | ทิศทาง |
|---|---|---|
| คลื่น A | ย้อนกลับ 3 คลื่น | สวนแนวโน้มหลัก |
| คลื่น B | ย้อนกลับ 3 คลื่น | ย้อนกลับส่วนใหญ่หรือทั้งหมดของ A |
| คลื่น C | แรงกระตุ้น 5 คลื่น | ทิศทางเดียวกับ A |

**จำนวนคลื่นย่อยรวม:** $3 + 3 + 5 = 11$

### 3.2 ประเภทของ Flat

#### 3.2.1 Regular Flat
คลื่น B ย้อนกลับ **ประมาณ 90%--105%** ของคลื่น A และคลื่น C สิ้นสุด **ประมาณที่** ระดับเดียวกับคลื่น A

#### 3.2.2 Expanded (Irregular) Flat
คลื่น B ย้อนกลับ **เกิน 100%** ของคลื่น A (เกินจุดเริ่มต้น) และคลื่น C ขยาย **เกินจุดสิ้นสุดของ** คลื่น A อย่างมาก

| ลักษณะ | ค่า |
|---|---|
| การย้อนกลับคลื่น B ของ A | 100%--138.2% (ปกติ 115%--127.2%) |
| คลื่น C เทียบกับ A | 127.2%--161.8% ของ A |
| ความลึกของการย้อนกลับรวม | ลึก |

**นี่เป็นประเภท Flat ที่พบบ่อยที่สุด โดยเฉพาะในตลาด Forex**

#### 3.2.3 Running Flat
คลื่น B ย้อนกลับ **เกิน 100% มาก** ของคลื่น A และคลื่น C **ไม่ถึง** จุดสิ้นสุดของคลื่น A สิ่งนี้สร้างรูปแบบที่การย้อนกลับสิ้นสุดใกล้ทิศทางแนวโน้มหลักมากขึ้น

**Running Flat มีความสำคัญเพราะบ่งชี้ว่าแนวโน้มหลักแข็งแกร่งมาก** การย้อนกลับแทบไม่ย้อนกลับเลย พบบ่อยที่สุดในฐานะคลื่น 2 ของคลื่น 3 ที่ขยายตัว

### 3.3 ความแตกต่างระหว่าง Flat กับ Zigzag

| ลักษณะ | Zigzag | Flat |
|---|---|---|
| โครงสร้างคลื่น A | แรงกระตุ้น 5 คลื่น | ย้อนกลับ 3 คลื่น |
| การย้อนกลับคลื่น B ของ A | ปกติ 38.2%--78.6% | ปกติ 90%--138.2% |
| ความลึกของการย้อนกลับ | ลึก, แหลมคม | ปานกลาง, ด้านข้าง |
| ตำแหน่งที่พบบ่อย | คลื่น 2 | คลื่น 4 |
| ลักษณะ | แหลมคมและเร็ว | ช้าและบดขยี้ |
| จำนวนคลื่นย่อย | 5-3-5 = 13 | 3-3-5 = 11 |

## 4. การย้อนกลับแบบง่าย: Triangle (3-3-3-3-3)

### 4.1 โครงสร้าง

Triangle ประกอบด้วย **ห้าคลื่น** กำหนดป้าย **A-B-C-D-E** แต่ละคลื่นแบ่งย่อยเป็น **สามคลื่น** (ย้อนกลับ)

**จำนวนคลื่นย่อยรวม:** $3 \times 5 = 15$

### 4.2 ประเภทของ Triangle

- **Contracting Triangle (พบบ่อยที่สุด)**: เส้นแนวโน้มเชื่อม A-C และ B-D หดเข้าหากัน แต่ละคลื่นสั้นกว่าคลื่นก่อนหน้า
- **Barrier Triangle**: เส้นแนวโน้มหนึ่งเส้นเป็นเส้นนอน คลื่น B และ D สิ้นสุดที่ระดับเดียวกันโดยประมาณ
- **Expanding Triangle (หายาก)**: เส้นแนวโน้ม **ขยายออก** แต่ละคลื่นยาวกว่าคลื่นก่อนหน้า
- **Running Triangle**: Triangle เอียงไปในทิศทางแนวโน้มหลัก คลื่น B เกินจุดเริ่มต้นของคลื่น A

### 4.3 ตำแหน่งที่ Triangle ปรากฏ

Triangle ปรากฏ **เฉพาะ** ในตำแหน่งเหล่านี้:

| ตำแหน่ง | บริบท | หมายเหตุ |
|---|---|---|
| คลื่น 4 | ภายในแรงกระตุ้น | ตำแหน่งที่พบบ่อยที่สุด |
| คลื่น B | ภายใน Flat หรือ Zigzag | พบบ่อย |
| คลื่น X | ภายในการย้อนกลับซับซ้อน | พบบ่อย |
| **ไม่เคยปรากฏใน** | คลื่น 2 หรือ คลื่น A | Triangle ไม่ปรากฏในตำแหน่งคลื่น 2 หรือ คลื่น A |

### 4.4 Thrust หลัง Triangle

หลังจาก Triangle เสร็จสมบูรณ์ ราคามัก **พุ่ง (Thrust)** อย่างรวดเร็วในทิศทางแนวโน้มหลัก Thrust มัก:

$$\text{Thrust} \approx \text{Range}_A \quad \text{(วัดจาก Apex)}$$

Thrust นี้เป็นหนึ่งในสัญญาณที่เทรดได้ดีที่สุดในการวิเคราะห์คลื่นอีเลียต

### 4.5 ความสัมพันธ์ Fibonacci

$$|B| \approx 0.618 \times |A|$$
$$|C| \approx 0.618 \times |B| \approx 0.382 \times |A|$$
$$|D| \approx 0.618 \times |C|$$
$$|E| \approx 0.618 \times |D|$$

แต่ละคลื่นมีขนาดประมาณ 61.8% ของคลื่นก่อนหน้า

### 4.6 การเทรด Triangle

**ชุดที่ 1: เทรด Post-Triangle Thrust**
```
CONDITION: รูปแบบ Triangle (A-B-C-D-E) เสร็จสมบูรณ์; คลื่น E สิ้นสุดใกล้เส้นแนวโน้ม B-D
ENTRY:     Buy/Sell ในทิศทางแนวโน้มเมื่อราคาทะลุเส้นแนวโน้ม A-C
STOP:      เกินจุดสิ้นสุดคลื่น E (ด้านตรงข้ามของ Triangle)
TARGET:    เป้า Thrust = ช่วงคลื่น A ฉายจากจุด Breakout
RR RATIO:  3:1 ถึง 5:1+ (ยอดเยี่ยม)
```

## 5. การย้อนกลับแบบซับซ้อน: Double และ Triple Zigzags

### 5.1 Double Zigzag (WXY)

เมื่อ Zigzag เดียวไม่บรรลุความลึกการย้อนกลับที่เพียงพอ ตลาดอาจสร้าง **Double Zigzag**: สอง Zigzag เชื่อมต่อด้วยคลื่นคั่น (คลื่น X)

**โครงสร้าง:** Zigzag - X - Zigzag

**การกำหนดป้าย:** W-X-Y (โดยที่ W และ Y เป็น Zigzag แต่ละอัน และ X เป็นตัวเชื่อมแบบย้อนกลับ)

### 5.2 Triple Zigzag (WXYXZ)

หายากยิ่งกว่า: สาม Zigzag เชื่อมต่อด้วยสองคลื่น X

**โครงสร้าง:** Zigzag - X - Zigzag - X - Zigzag

### 5.3 ความสัมพันธ์ Fibonacci

$$|Y| \approx |W| \quad \text{(ความเท่าเทียมพบบ่อยที่สุด)}$$

$$|Y| = r \times |W| \quad \text{where } r \in \{0.618, 1.000, 1.618\}$$

**คลื่น X มักย้อนกลับ 38.2%--61.8% ของคลื่น W**

## 6. การย้อนกลับแบบซับซ้อน: Double และ Triple Threes (WXY / WXYXZ)

### 6.1 คำจำกัดความ

**Double Three** หรือ **Triple Three** เป็นชุดรวมด้านข้างของสองหรือสามรูปแบบย้อนกลับ เชื่อมต่อด้วยคลื่น X ต่างจาก Double/Triple Zigzag (ที่ลึก) ชุดรวมเหล่านี้เป็นแบบ **ด้านข้าง** และสร้างการย้อนกลับที่กินเวลาและน่าหงุดหงิดที่สุด

### 6.2 ชุดรวมที่พบบ่อย

| คลื่น W | คลื่น X | คลื่น Y | ความถี่ |
|---|---|---|---|
| Zigzag | ย้อนกลับ | Flat | พบบ่อย |
| Zigzag | ย้อนกลับ | Triangle | พบบ่อย |
| Flat | ย้อนกลับ | Zigzag | ปานกลาง |
| Flat | ย้อนกลับ | Triangle | ปานกลาง |
| Flat | ย้อนกลับ | Flat | น้อยกว่า |

**กฎสำคัญ:** Triangle สามารถปรากฏเป็นรูปแบบ **สุดท้าย** เท่านั้นในชุดรวม (คลื่น Y ใน WXY หรือ คลื่น Z ใน WXYXZ)

### 6.3 การเทรดชุดรวม (Combinations)

ชุดรวมเป็นการย้อนกลับที่ **ยากที่สุดในการเทรด** เพราะ:
- ขยายตัวด้านข้างเป็นเวลานาน
- แต่ละครั้งที่รูปแบบดูเหมือนเสร็จ คลื่น X ใหม่และการย้อนกลับใหม่จะเริ่มต้น
- มักหลอกเทรดเดอร์ที่พยายามเทรด Breakout เร็วเกินไป

**แนวทางที่ดีที่สุดสำหรับ AI:**
```
1. ระบุ ว่าชุดรวมกำลังก่อตัว (คลื่น W เสร็จ, คลื่น X กำลังพัฒนา)
2. รอ ให้ชุดรวมเสร็จสมบูรณ์แทนที่จะเทรดภายใน
3. ติดตาม Triangle เป็นส่วนประกอบสุดท้าย (ส่งสัญญาณเสร็จสมบูรณ์ใกล้แล้ว)
4. เข้า เฉพาะหลังจาก Post-Combination Thrust ยืนยันทิศทางแนวโน้มใหม่
5. ใช้ Triangle Thrust หรือ คลื่น C ของส่วนประกอบสุดท้ายเป็นจุดเข้า
```

## 7. Running Corrections และรูปแบบผิดปกติ

### 7.1 Running Flat (โดยละเอียด)

ใน Running Flat คลื่น B เกินจุดเริ่มต้นของคลื่น A และคลื่น C ไม่ถึงจุดสิ้นสุดของคลื่น A

**ความสำคัญ:** Running Flat บ่งชี้ความแข็งแกร่งสุดขั้วในทิศทางแนวโน้มหลัก การย้อนกลับแทบไม่ย้อนกลับเลย

### 7.2 Expanded Flat (B-Wave Traps)

Expanded Flat อันตรายเป็นพิเศษสำหรับเทรดเดอร์เพราะ:
1. Thrust ของคลื่น B ที่เกินจุดเริ่มต้นคลื่น A ดูเหมือนแนวโน้มต่อเนื่อง
2. เทรดเดอร์เข้าในทิศทางแนวโน้ม คาดว่าจะเป็น Breakout
3. คลื่น C จากนั้นกลับตัวอย่างแหลมคม ตัด Stop Out เทรดเดอร์เหล่านั้นและเกินจุดสิ้นสุดคลื่น A

**สำหรับการตรวจจับด้วย AI:** กุญแจคือการรับรู้ว่าคลื่น A มีโครงสร้าง 3 คลื่น (ไม่ใช่ 5 คลื่น) หากการเคลื่อนที่สวนแนวโน้มเริ่มต้นเป็นรูปแบบ 3 คลื่น Expanded Flat เป็นความเป็นไปได้ที่แข็งแกร่ง และ "Breakout" (คลื่น B เกินจุดเริ่มต้น) เป็นกับดัก

## 8. ระดับ Fibonacci Retracement สำหรับรูปแบบย้อนกลับ

### 8.1 ตารางการย้อนกลับหลัก

| ประเภทการย้อนกลับ | การย้อนกลับทั่วไปของแรงกระตุ้นก่อนหน้า | ระดับ Fibonacci |
|---|---|---|
| **Zigzag** | ลึก | 50%, 61.8%, 78.6% |
| **Double Zigzag** | ลึกมาก | 61.8%, 78.6%, 88.6% |
| **Regular Flat** | ปานกลาง | 38.2%, 50% |
| **Expanded Flat** | ลึก (จากการขยายคลื่น C) | 50%, 61.8%, 78.6% |
| **Running Flat** | ตื้น | 23.6%, 38.2% |
| **Contracting Triangle** | ตื้นถึงปานกลาง | 23.6%, 38.2%, 50% |
| **Combination (WXY)** | ปานกลาง | 38.2%, 50% |

### 8.2 ความสัมพันธ์ Fibonacci ภายใน

**ภายใน Zigzag:**
$$|B_{\text{zigzag}}| = r \times |A_{\text{zigzag}}| \quad r \in \{0.382, 0.500, 0.618, 0.786\}$$
$$|C_{\text{zigzag}}| = e \times |A_{\text{zigzag}}| \quad e \in \{0.618, 1.000, 1.272, 1.618\}$$

**ภายใน Flat:**
$$|B_{\text{flat}}| \approx |A_{\text{flat}}| \quad \text{(ประมาณ 90%-138.2%)}$$
$$|C_{\text{flat}}| = e \times |A_{\text{flat}}| \quad e \in \{1.000, 1.272, 1.618, 2.618\}$$

**ภายใน Triangle:**
$$|B| \approx 0.618 \times |A|$$
$$|C| \approx 0.618 \times |B|$$
$$|D| \approx 0.618 \times |C|$$
$$|E| \approx 0.618 \times |D|$$

## 10. โมเดลทางคณิตศาสตร์

### 10.1 โมเดลราคาคลื่นย้อนกลับ

**โมเดล Zigzag (การย้อนกลับขาลงในแนวโน้มขาขึ้น):**

$$P_A = P_0 - \Delta_A \quad (\Delta_A > 0)$$

$$P_B = P_A + r_B \cdot \Delta_A \quad \text{where } 0 < r_B < 1$$

$$P_C = P_B - e_C \cdot \Delta_A \quad \text{where } e_C > 0 \text{ and } P_C < P_A$$

**โมเดล Flat (การย้อนกลับขาลงในแนวโน้มขาขึ้น):**

$$P_A = P_0 - \Delta_A$$

$$P_B = P_A + r_B \cdot \Delta_A \quad \text{where } r_B \approx 1.0 \text{ (can exceed 1.0)}$$

$$P_C = P_B - e_C \cdot \Delta_A \quad \text{where } e_C \geq 1.0$$

**โมเดล Triangle:**

$$P_{n+1} = P_n + (-1)^n \cdot k^n \cdot \Delta_A \quad \text{where } k \approx 0.618 \text{ (contraction factor)}$$

สำหรับ $n = 0, 1, 2, 3, 4$ (คลื่น A ถึง E)

### 10.2 ตัวประมาณความลึกการย้อนกลับ

| ประเภทที่คาดหวัง | ตัวคูณความลึก $d$ | ช่วง |
|---|---|---|
| Running Flat | 0.146--0.382 | ตื้น |
| Triangle | 0.236--0.382 | ตื้น |
| Regular Flat | 0.382--0.500 | ปานกลาง |
| Zigzag | 0.500--0.618 | ลึก |
| Expanded Flat | 0.500--0.786 | ลึก |
| Double Zigzag | 0.618--0.886 | ลึกมาก |

### 11.2 ตารางการตัดสินใจเทรดช่วงย้อนกลับ
