## 7. Key discoveries / corrections today (all doc-verified)
1. **"SQL-warehouse compute affinity forces workspace membership" = FALSE.** *"Compute access is always granted by the publisher's credentials"* → **data identity = viewer, compute identity = publisher.**
2. **"We can't GRANT to their groups without adding them to coredata" = MISCONCEPTION.** A **UC GRANT is a metastore-level data privilege**; **workspace assignment is a separate operation**. *"Members inherit permissions from all account groups… regardless of whether the group is assigned to the workspace."* **Nobody joins coredata.**
   - ⚠️ *Real kernel underneath:* if their groups are **workspace-local (legacy)** they cannot be granted UC data access **from anywhere** → fix = an **account admin creates account groups** (Entra sync / Automatic Identity Management). That is **not** a coredata change.
   - ⚠️ `CREATE GROUP` in Databricks SQL creates **workspace-local** groups — **do not use it**.
3. **A UC catalog has NO home workspace.** *"All catalogs are accessible by default from any workspace attached to the same metastore."* → departmental may already see it. Exception: an **`ISOLATED`** catalog, or the auto-created **default workspace catalog (bound by default)**.
4. **Publish mode is the #1 field failure.** Default **"Share data permissions"** = publisher's credentials → **row filters evaluate as the publisher → every team sees every row** (silent). Must use **"Individual data permissions"** (`embed_credentials: false`).
5. **"AI/BI is not secure" = FALSE as stated.** No anonymous access; viewer must be a registered Entra/Databricks identity **and** be explicitly granted. GA vs Preview; **1 way to misconfigure vs ≥4 for a Databricks App** (whose token-forwarding bug silently falls back to the app's service principal = full data).
6. **Genie is NOT dead** (an earlier note said so) — it's blocked only for *account users without workspace membership*. **Inside their own workspace they are workspace users → Genie works.**
7. **Terminology (kills Sarunya's "new feature" question):** **Lakeview = AI/BI Dashboard** (same product; API still `/lakeview`). **Legacy "Databricks SQL Dashboard" = RETIRED.** **Databricks One → renamed Genie → "Genie One"**. **Genie Space → Genie Agent** (Jul 2026). **Delta Sharing → OpenSharing**.
8. **`is_account_group_member()`, never `is_member()`** — the latter silently returns FALSE for anyone not in the *current* workspace = your whole audience.
9. **`GRANT EXECUTE` on the row-filter function** is the most-forgotten grant.
10. **Row filters cannot be applied to a VIEW** → per-team views are an *alternative* to RLS, not a complement.
